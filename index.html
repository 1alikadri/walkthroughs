Sublist3r Walkthrough

1. Introduction to Sublist3r

Description: Sublist3r is a Python-based tool designed to enumerate subdomains by leveraging major search engines like Google, Bing, and Yahoo. It assists penetration testers or security researchers during the reconnaissance stage by revealing subdomains, often identifying hidden or less secure web applications.

Use Cases: Typically used in the reconnaissance phase of a penetration test to map out the subdomains of a target.

2. Prerequisites

System Requirements: Sublist3r is a lightweight tool and only requires basic system resources.

Dependencies: Requires Python 2.7 and additional Python libraries. Kali Linux usually has Python installed, but certain libraries may need to be added.

3. Installation Steps

  1. Step 1: Refresh Package List 
    sudo apt update
  2. Step 2: Clone the Repository 
    git clone https://github.com/aboul3la/Sublist3r.git
  3. Step 3: Switch to the Sublist3r Directory 
    cd Sublist3r
  4. Step 4: Install Dependencies 
    sudo apt install python-pip
pip install -r requirements.txt

4. Basic Configuration

After installation, Sublist3r is ready to use. Ensure a stable internet connection, as the tool queries online services.

5. Basic Usage Overview

Basic Usage Command: To enumerate subdomains, use:

python sublist3r.py -d example.com

Options:

  • -d: The domain to be enumerated.
  • -b: Brute force subdomains for deeper searches.
  • -p: Scan specified ports for each subdomain.
  • -v: Verbose mode to display progress in real-time.

Example Command:

python sublist3r.py -d example.com -b -p 80,443 -v

6. Advanced Features

Brute-forcing: Use -b flag to enable brute-forcing. This increases runtime but reveals additional subdomains.

Port Scanning: Use the -p option to scan specific ports (e.g., 80, 443).

Save Output to a File: Add > filename.txt to save results to a file.

7. Troubleshooting

Missing Dependencies: Install missing Python packages:

pip install requests argparse dnspython

Python Version Errors: If Kali defaults to Python 3, run Sublist3r using python2 instead of python.

8. Real-World Example

Scenario: Reconning the domain example.com to enumerate its subdomains and check for open HTTP/HTTPS ports.

Command:

python sublist3r.py -d example.com -b -p 80,443 -v > example_subdomains.txt

This command will enumerate subdomains for example.com, brute-force additional subdomains, scan ports 80 and 443, and save results to example_subdomains.txt.
